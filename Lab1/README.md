# Lab 1: Exercises from "Graphical Data Analysis with R"

## Task

Complete one exercise from each chapter (3–11) of "Graphical Data Analysis with R" (Unwin). Variant 5 (Group 4).

### Packages

tidyverse, ggplot2, patchwork, GGally, ggpcp, ggalluvial, ggmosaic, extracat, vcd, vcdExtra, naniar, VIM, rayshader

## Results

- **Chapter 3** — Histograms and densities: distribution of educational funding (Zuni dataset)
- **Chapter 4** — Bar charts: occupational mobility of fathers and sons
- **Chapter 5** — Mosaic plots: gender discrimination in bank hiring
- **Chapter 6** — Scatterplots: body composition analysis (Bodyfat)
- **Chapter 7** — Parallel coordinates: airline flight delays
- **Chapter 8** — Missing data: visualizing missingness patterns (Pima Indians)
- **Chapter 9** — Pair plots: heights of fathers and sons (Pearson heights)
- **Chapter 10** — Multivariate methods: diamond characteristics analysis
- **Chapter 11** — Time series: Bundesliga goals
