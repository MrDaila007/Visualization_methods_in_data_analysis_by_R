# Lab 10: Quarto Report (Portfolio)

## Task
Prepare a Quarto report including laboratory works and tests.

## Details
- Download hello.qmd: https://quarto.org/docs/get-started/hello/rstudio.html

## Tools
- quartify - https://ddotta.github.io/quartify/ (R package to convert R scripts into .qmd)
- Convert R Script to Quarto: https://quartify.lab.sspcloud.fr/
- rd2qmd - https://github.com/eitsupi/rd2qmd (Rd-to-Quarto Markdown converter)
- ripper - https://quarto.thecoatlessprofessor.com/ripper/ (extracts code blocks from documents)

## Format
qmd, png, html
