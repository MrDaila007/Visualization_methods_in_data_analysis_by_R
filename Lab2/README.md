# Lab 2: Automated Graphical Exploratory Data Analysis (autoGEDA)

## Task

Perform automated graphical exploratory data analysis (autoGEDA) using specialized packages.

### Packages

dlookr, brinton, plotluck, gtExtras

### Data

TidyTuesday datasets

## Results

- Applied **dlookr** for automatic data diagnostics and report generation
- Used **brinton** for automatic chart type selection
- Built visualizations with **plotluck** for rapid prototyping
- Created a manual autoGEDA approach with automatic variable type detection and chart selection
- Generated summary tables using **gtExtras**
