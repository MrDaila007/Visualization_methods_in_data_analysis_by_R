# Lab 3: Graphical Exploratory Data Analysis (GEDA)

## Task

Perform graphical exploratory data analysis (GEDA). Formulate analytical questions of each type (elementary and synoptic).

### Data

Palmer Penguins (TidyTuesday)

## Results

### Elementary Questions
- **Identify** — determining specific values in the data
- **Locate** — finding where values are positioned
- **Compare** — comparing variables across groups
- **Rank** — ranking species by characteristics
- **Connect** — relationships between variables
- **Distinguish** — distinguishing features of species

### Synoptic Questions
- **Group** — clustering by morphological traits
- **Correlate** — dependencies between bill length and body mass
- **Trends** — systematic changes over years
- **Cycles** — periodic patterns
- **Outliers** — detecting anomalous observations
- **Features** — characteristic distribution properties
