# Lab 4: Build a Heat Map

## Task

Build a heatmap. Visualize a dataset as a table (heatmap) using various packages.

### Packages

ComplexHeatmap, pheatmap, ggplot2 (geom_tile), gplots, gt, kableExtra, DT, reactable, formattable

### Data

airquality (TidyTuesday)

## Results

- Built 5 heatmap variants:
  1. **ggplot2** `geom_tile()` — basic heatmap
  2. **pheatmap** — with dendrograms and clustering
  3. **ComplexHeatmap** — extended annotated heatmap
  4. **gplots** `heatmap.2()` — classic heatmap
  5. **stats** `heatmap()` — base R heatmap
- Explored color interpolation methods
- Created interactive tables using gt, kableExtra, DT, reactable, formattable
