# Lab 5: Create the Ugliest Plot Possible!

## Task

Create the ugliest plot possible! Study 11 types of visual misleaders that lead to incorrect conclusions.

### Resources

- [Misinformed by Visualization](https://leoyuholo.com/bad-vis-browser/)
- [VisLies](https://www.vislies.org/2024/)
- Alberto Cairo "How Charts Lie"
- [Geckoboard Data Fallacies](https://www.geckoboard.com/best-practice/statistical-fallacies/)

## Results

- Created the **"ugliest plot ever"** violating every visualization principle: acid colors, Comic Sans, 3D effects, flipped labels, excessive elements
- Analyzed visual misleaders with **before** and **after** examples:
  - Truncated Y-axis
  - Incorrect proportions
  - Dual axes
  - Cherry-picking data
  - Inconsistent scales
- Demonstrated how to fix each type of misleader
