# Lab 6: ggplot Crossword Puzzle + Visualization Recreation

## Task

1. Solve the ggplot2 crossword puzzle by [Georgios Karamanis](https://karaman.is/blog/2020/12/ggplot-crossword)
2. Recreate a visualization as closely as possible. Record a gif with camcorder.

### Data

data-I2Rsw.csv — "How many people get sick from foodborne illnesses?" (Group 4, USAFacts)

## Results

- Solved the crossword on ggplot2 functions and parameters
- Recreated the USAFacts foodborne illness visualization as a **dumbbell chart**
- Built alternative visualizations of the same data:
  - Grouped bar chart
  - Slope chart
  - Heatmap
- Used the **camcorder** package to record the visualization creation process
