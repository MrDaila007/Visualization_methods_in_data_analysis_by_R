### Redesign
- Redesigned the New Zealand data table using visual channels
- Applied principles: avoid unnecessary chart junk, maximize data-ink ratio

### Channel Effectiveness
- Magnitude channels: position (most effective) → length → area → color luminance
- Identity channels: spatial region → color hue → shape
