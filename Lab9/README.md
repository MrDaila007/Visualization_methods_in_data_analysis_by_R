### Color
- Color spaces: RGB, HSL, HCL, CIELAB
- Palette types: sequential, diverging, qualitative
- **Color accessibility**: color blindness simulation (deuteranopia, protanopia, tritanopia)
- Tools: colorspace `specplot()`, colorBlindness, paletteer
- Principles: luminance variation for accessibility, avoid rainbow palettes
